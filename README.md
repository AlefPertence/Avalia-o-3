# Sistema de Manipulação de Grafos em C
# Descrição

Este projeto implementa um grafo não direcionado utilizando uma matriz de adjacência. O programa permite criar e manipular conexões entre vértices por meio de um menu interativo no terminal.

# Funcionalidades
- Limpar o grafo
- Adicionar arestas
- Remover arestas
- Exibir a matriz de adjacência
- Calcular o grau dos vértices
- Listar os vértices adjacentes

# Estruturas de Dados Utilizadas
- Vetor bidimensional (matriz de adjacência)
- Grafos não direcionados
- Estruturas de repetição
- Funções para modularização do código

# Tecnologias
- Linguagem C
- Biblioteca stdio.h
- Biblioteca stdlib.h
